# D – Disability (Neurological) Assessment Guidelines
Context: Emergency call transcript analysis

## Purpose
Assess neurological status and level of consciousness.

## What to Evaluate
- consciousness
- orientation and responsiveness
- seizures
- sudden neurological deficits

## Caller Language Indicators

### Normal
- alert and responsive
- answers appropriately
- oriented

### Impairment
- confused
- slow responses
- disoriented
- very drowsy

### Critical Indicators
- unconscious
- seizure activity
- not responding
- sudden speech problems
- sudden weakness
- facial droop

## Assessment Rules
- responsiveness is key indicator
- caller uncertainty must be documented

## Red Flags
- unconsciousness
- seizures
- sudden neurological deficit

## Severity Scale
0 = no information given or no conclusions possible
1 = no impairment suspected
2 = significant impairment  
3 = critical neurological state

## Missing Information
- Is the patient awake?
- Responding normally?
- Any seizure activity?

## Output Reminder
Use functional descriptions only.
